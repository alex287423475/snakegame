# 🐍 贪吃蛇游戏

一个使用纯 HTML、CSS 和 JavaScript 开发的现代化贪吃蛇游戏。

## 功能特点

- ✨ 精美的现代化 UI 设计
- 🎮 流畅的游戏体验
- 📊 得分系统和最高分记录（本地存储）
- ⏸️ 暂停/继续功能
- 🎯 碰撞检测（墙壁和自身）
- 📱 响应式设计，支持移动设备
- ⌨️ 键盘控制（方向键）

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需要安装 http-server)
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 游戏操作

- **方向键 ↑ ↓ ← →**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **开始游戏**：点击按钮开始游戏
- **暂停**：暂停当前游戏
- **重置**：重置游戏状态

## 游戏规则

1. 使用方向键控制蛇的移动
2. 吃到红色食物可以增长身体并得分（+10分）
3. 撞到墙壁或自己的身体会游戏结束
4. 尽可能获得更高的分数！

## 技术栈

- HTML5 Canvas
- CSS3 (渐变、动画、响应式设计)
- 原生 JavaScript (ES6+)

## 浏览器支持

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
